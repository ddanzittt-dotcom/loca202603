## 1. 구조 제안 — "챕터형" 가이드

현행은 어느 주제를 골라도 그 스텝부터 **끝까지 한 줄로** 재생된다("공유하는 법"을 골라도 마지막 1스텝만 나옴).
개편안: **주제별 챕터**로 재편 — 메뉴에서 주제를 고르면 그 챕터만 재생하고 닫힌다.

```
자동 재생 (첫 입장 1회 · 로그인 첫 진입 1회) ──▶ [챕터 0] 전체 사용법 (5스텝, 짧게 유지)

로카냥 메뉴 "뭐가 궁금해?" ──▶ 챕터 선택
  📖 처음부터 전체 사용법      → 챕터 0 intro     (5스텝)
  🧺 장소 채집하는 법          → 챕터 1 collect   (5스텝)
  🗂️ 카드에 기록 남기는 법     → 챕터 2 binder    (6스텝)
  🗺️ 지도 만드는 법            → 챕터 3 mapmaking (6스텝)
  📣 공유하고 함께 만드는 법    → 챕터 4 share     (5스텝)
  🧭 탐색 탭 안내              → 챕터 5 explore   (4스텝)
  🐾 산책 모드                 → 챕터 6 walk      (4스텝)
```

> 내 수첩 챕터(§10)는 v1.1 결정으로 **미포함(보류)** — 카피는 아래에 보존, 추후 추가 시 챕터 id `notebook`.

- 자동 재생 코스는 **지금처럼 짧게(5스텝)** 유지 — 첫인상에서 이탈 방지. 상세는 전부 메뉴 챕터로.
- 대화창 UI(포켓몬식 타자기·탭 진행·건너뛰기)는 그대로 재사용. 스텝 도트는 챕터 내 진행만 표시.
- 챕터 마지막 스텝 버튼: 챕터 0은 "시작하기!", 나머지는 "알겠어!".
- (선택) 챕터 끝에서 "다른 주제도 볼래?" 로 메뉴 복귀 — v1은 생략 가능.

## 2. 카피 원칙 (브랜딩 문서 준수)

- **'채집=양념' 문법**: 버튼 언급 = **담다**("'카드로 담기'를 눌러"), 축적 결과 = **모으다**, 명사·판정·놀이표면 = **채집**(SPOT/NEW FIND, 산책 모드, "좋은 채집 되길").
- 수집가의 어휘: 모으다·담다·꺼내다·꽂다·채우다. 금지: 관리하다·등록하다·운영하다.
- 스텝 카드(제목·설명) = **존댓말**, 로카냥 말풍선 = **반말 + 이따금 '냥/냐옹'**. (현행 톤 유지)
- UI 라벨은 실제 문구를 따옴표로 정확히 인용: '카드로 담기' '이 위치로 담기' '새 지도' 'N곳으로 지도 만들기' '길 완성' '영역 완성' '검색·탐색에 공개' '함께 만들기' '내 수첩' 등.
