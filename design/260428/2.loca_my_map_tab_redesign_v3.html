설계 노트 · 내 지도 탭 리디자인

  • 헤더 압축 — 4단(LOCA 로고 + H1 + 부제 + 텍스트 버튼 2개) → 1단으로. LOCA 로고는 제거(홈 탭 정체성 신호와 중복), 부제도 제거. 좌측 H1 "내 지도", 우측 아이콘 그룹.
  • 새 지도 만들기 = primary 아이콘 버튼 — A 화면(지도 탭) 우측에 지도 가져오기 → 새 지도(+) → 알림 3개 아이콘. "새 지도 만들기"는 primary 오렌지 배경의 + 버튼으로 노출하여 시각적으로 가장 강한 액션임을 표시. 새 지도 만들기 ≠ FAB: FAB는 현재 지도 안에서 마커(장소/경로/범위) 추가, 헤더의 +는 새로운 지도 컨테이너 생성. 두 액션의 결과물이 다르므로 위치도 분리. B 화면(장소 탭)에는 노출 안 함(장소 탭에서 "새 지도 만들기"는 맥락에 안 맞음, 가져오기/알림만 유지).
  • 탭 스위처 segmented 스타일 — 이전엔 좌우로 분리된 알약 버튼이라 활성/비활성 차이가 너무 강했음. 흰색 컨테이너 안에 secondary 그린 슬라이더 형태로 변경하여 iOS segmented control 패턴 적용. 우측에 카운트 노출.
  • 지도 카드: 한 카드 = 한 지역 컬러 — DESIGN.md §2.5 Regional Color System 적용. 카드 배경을 region pale 톤(#E8BCAD 서울, #ABC6DC 부산, #BCD3BD 세종)으로 깔고, blob 3개를 살짝 흩뿌려 부드러운 분위기. 단색 회색이었던 이전 시안 대비 지역감이 시각적으로 살아남.
  • 카드 정보 통일 — 좌상단 region 칩 + 우상단 visibility 뱃지(나만 보기 / 링크 공유 중 / 프로필 공개) 위치 고정. 하단은 그라데이션 오버레이 위에 명조체 제목 + 장소·경로·범위 카운트 3종(타입 아이콘 + 숫자) + 수정일. 우측에 더보기(··) 버튼.
  • 3종 마커 카운트 노출 — 보고서 P0(용어 분리) 결과 적용. 핀 아이콘 + 폴리라인 + 점선 사각형 SVG. 카운트가 0인 타입은 표기하되 흐리게(예: 구역 0). 사용자가 자신의 지도 구성을 한눈에 파악.
  • 수정일 텍스트 짧게 — "오늘 13:24" 같은 절대시간 → 오늘 / 어제 / 3일 전처럼 상대시간. 모바일 카드 좁은 영역에서 가독성 우위.
  • 장소 탭: 3단 필터 시스템 — 이전 2단 빽빽한 칩 스택 → 그룹 라벨 + 가로 스크롤 3단. 좌측에 회색 그룹 라벨(지도 / 타입 / 기록)을 두어 각 칩 row의 의미를 명확히. 빽빽한 칩 클러스터의 판단 비용 절반 감소.
  • "전체" 칩 활성 상태 명확화 — 이전엔 "전체 52"가 두 줄에 모두 active 상태로 있어 "둘 다 활성?"으로 헷갈렸음. v2에선 각 row마다 독립적인 active 상태(현재 모두 "전체" 선택 상태). 한 row 내 칩은 단일 선택이라는 시각 신호 강화.
  • 타입 칩에 컬러 닷 — ● 장소 / ● 경로 / ● 범위. DESIGN.md §2.3 마커 컬러를 칩에 동기화. 사용자가 색을 보고 빠르게 식별.
  • 장소 row 컴팩트화 — 이전 row 높이 ~70px → ~58px. 우측 "장소" 라벨을 제거하고(필터 칩에 이미 정보 있음), 대신 소속 지도 이름 + 메모/사진 개수를 메타 라인에 노출. 한 화면에 보이는 row 수 30%↑.
  • 핀 아이콘 박스에 이모지 노출 — 이전은 단조로운 오렌지 핀만 반복 → 사용자 선택 이모지(☕ 🥖 🌸) 노출. 시각 다양성 + 사용자가 의도한 카테고리감 즉시 인지. 경로/범위는 SVG로 유지(이모지 없음).
  • "새 장소" 안티패턴 처리 — DESIGN.md §14 Don'ts에 명시된 "새 장소를 기본 이름으로 방치 금지" 적용. 장소 이름이 비어있으면 이탤릭 회색 톤으로 "이름 없는 장소 · 직접 입력하기"라고 표기. 사용자에게 입력 유도.
  • 장소 row 우측 chevron — 우측에 ›(chevron) 아이콘 추가. 탭 가능 어포던스 + 상세 진입점이 명확해짐.
  • 변경 안 한 것 — 정보 우선순위, 탭 구조(지도/장소), 검색 위치, 탭바, FAB 위치는 그대로. 옷만 다림질한 수준이라 개발팀이 받아서 스타일만 교체 가능.
