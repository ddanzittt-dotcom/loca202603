02 / MARKERS

핀 · 경로 · 영역 — 다양한 표현 옵션

하나만 고를 필요 없음. 데이터/맥락에 따라 조합 가능.

PIN핀 — 6가지 표현

A. 도트 (기본)
현재 LOCA 시그너처. 깔끔, 멀리서도 식별
B. 번호 핀
가이드북·코스 순서 명확. 발행 지도 추천
C. 드롭 핀
정확한 위치 표시. 구글맵 스타일
D. 카테고리 아이콘
☕📚🌳 카페·서점·산책 한눈에 구분
E. 사진 핀
실제 장소 썸네일. 인스타그래머블, 시각적 강함
F. 이모지 핀 (개인 큐레이션)
친근, 사용자가 직접 선택. 개인 지도 추천
추천 조합 · 개인 지도는 A 도트 + F 이모지(사용자 선택), 공유/발행 지도는 B 번호 핀 순서, 추천/큐레이션은 E 사진 핀으로 임팩트.

PATH경로 — 4가지 표현

A. 손맛 곡선 + 화살표 (기본)
v7 시그너처. 시작■ 화살▶ 끝◯
B. 점선
계획 중인 코스, 추천 경로 (미확정)
C. 노드 강조 (waypoints)
중간 정거장이 의미 있을 때. 등산·자전거
D. 그라디언트 (시간/난이도)
아침→저녁, 평지→오르막 등 변화 표현

AREA영역 — 4가지 표현

A. 채움 + 사선 hatch (기본)
v7 시그너처. 한 시절의 결, 라벨 포함
B. 외곽선 강조
행정구역, 명확한 경계 표현
C. 히트맵 (밀도)
자주 가는 영역, 핀이 많이 모인 동네
D. 점선 외곽
모두의 지도 제안 영역 / 잠정 영역 (개인 지도 X)

조합 가이드 — 어떤 지도엔 어떤 표현?

개인 다이어리 지도
핀 A(도트) 또는 F(이모지) · 경로 A(손맛) · 영역 A(채움+hatch)
발행 큐레이션 (가이드북)
핀 B(번호) · 경로 A(손맛+화살표) · 영역 B(외곽선)
맛집 큐레이션
핀 D(카테고리 아이콘) · 영역 C(히트맵 — 자주 가는 곳)
이벤트 / 스탬프 투어
핀 B(번호) + 체크인 시 Moss 채움 · 경로 A · 영역 B
함께 만드는 지도 (커뮤니티)
제안 핀 (대시 외곽) · 경로 B(점선) · 영역 D(점선 외곽)
공공데이터 폐점 / 이전 (모두의 지도 전용)
Dusk 톤 + 점선 외곽 + 55~70% opacity. 개인 지도/장소에는 적용하지 않음 — 사용자가 폐점한 가게도 그냥 "내 기록"으로 남길 수 있어요.
03 / QA + DATA

QA 체크리스트 · 데이터 모델 · 마이그레이션

"사라진은 내 지도에 X, 모두의 지도에만 ○" 원칙 검증.

★핵심 원칙

사라진 / DISAPPEARED / 폐점 / 이전됨 / 임시 휴업 / 확인 필요는 오직 모두의 지도(공공데이터 기반)에서만 사용.

내 지도 · 내 장소 · 내 프로필 · 개인 큐레이션에는 절대 적용하지 않음. 사용자는 폐점한 가게도 그냥 "내 기록"으로 남길 수 있다.

DATA공공데이터 상태 모델

// 모두의 지도 전용
publicPlace.status =
  "open" | "closed" | "moved" |
  "temporarily_closed" | "unknown"

publicPlace.statusSource =
  "public_data" | "user_report" |
  "admin_verified"

publicPlace.statusUpdatedAt = string

// 신규 제안은 별도
feature_change_requests.action =
  "insert" | "update" | "delete" |
  "status_update"

// 상태 수정 요청
payload = {
  publicStatus,
  statusNote,
  evidencePhotos
}

// 다중 외부 링크
profileLinks: [{
  id, type, label, url, order
}]
// type: "naver" | "youtube" |
//   "instagram" | "website" | "custom"
중요 · 개인 feature에는 status 필드를 추가하지 않음. 개인 지도는 status-free.

DELTA이번 통합에서 변경된 것

홈 · 검색 아이콘 추가, "전체 →" → "탐색 →", "사라진 가게" → "공공데이터로 본 변화 (모두의 지도)"
지도 탭 · 필터 "사라진 3" 제거 → "이벤트" 추가. "사라진" 배지/카드 모두 제거
장소 탭 · DISAPPEARED 항목 제거. 기록 카운트(사진/음성/메모) 추가
모두의 지도 · 필터 = 운영 중/사라진/이전됨/제안. 공공데이터 상태 도입
모두의 지도 핀카드 · "편집" 제거, "수정 요청"/"상태가 달라요" 추가, 권한 분기
★ 모두의 지도 · 폐점 카드 (NEW) · 09번 신규. Dusk + dashed + 안내 + 추억 메모
내 프로필 · "발행한 지도" → "보여주는 지도"
다른 사람 프로필 · 메시지 버튼 제거, "지도/참여" 탭, 공유로 대체
핀 표현 가이드 · "사라진 (모든 지도)" → "공공데이터 폐점/이전 (모두의 지도 전용, 개인 X)"

QA실제 적용 시 검증 항목

✓지도 탭에 "사라진" 필터가 없는지
✓내 지도 카드에 "사라진" 배지가 없는지
✓내 장소 탭에 "DISAPPEARED" 항목이 없는지
✓개인 지도/장소에서 폐점/사라짐 시스템 상태가 노출되지 않는지
✓모두의 지도에서 공공데이터 폐점/이전 상태가 표시되는지
✓모두의 지도에서 "제안"과 "폐점/이전"이 시각적으로 구분되는지 (Ember vs Dusk)
✓모두의 지도에서 상태 수정 요청이 가능한지
✓새 지도에 첫 기록 추가 플로우 정상
✓기존 지도에 핀 추가 플로우 정상
✓경로/영역 그리기 진입점 노출
✓장소 탭에서 핀/경로/영역 필터 동작
✓프로필에 지도 올리기와 발행 상태 분리
✓다른 사람 프로필에서 편집/지도 올리기/메시지 버튼이 보이지 않는지
✓모두의 지도에서 남의 장소 가져오기 (개인 지도엔 status 복사 안 됨)
✓모두의 지도에서 수정 요청 폼 동작
✓키보드 표시 중 + FAB 숨김
✓공유지도 viewer에서 + FAB 숨김
✓feature 편집 시트/모달 열림 중 + FAB 숨김
