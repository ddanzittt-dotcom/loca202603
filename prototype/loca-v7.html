03 / MAP RENDERING

지도 가시성 — 핀 · 경로 · 영역

핀은 Ember로 도드라지게, 경로는 Moss + 방향 화살표, 영역은 채도 올리고 라벨 강조. 사라진 요소는 더 선명히 구분.

A핀 (장소) — Ember 도드라짐

10px 도트 + 흰 헤일로로 지도 위에서 즉시 인식. 선택 시 외곽 링·중심 흰점 추가. 그림자로 떠 있는 느낌.

기본 핀
12px 흰 헤일로 + 6px Ember 도트
선택된 핀
2단 외곽 링 + 중심 흰점
클러스터
잉크 + 숫자 — 핀과 시각 구분
사라진 핀
Dusk + 55% opacity

B경로 (Path) — Moss + 화살표

2.8px Moss + 흰 헤일로 5px로 지도와 분리. 시작은 사각, 끝은 원, 중간엔 방향 화살표.

기본 경로
시작(■) · 화살표(▶) · 끝(◯)
사라진 경로
Dusk + 점선 + 75% opacity
핀은 Ember(빨강 계열), 경로는 Moss(녹색)로 색분리. 한눈에 "장소 vs 흐름" 구분됨.

C영역 (Area) — 채도 + 라벨

13% Moss 채움 + 1px 외곽선 + 사선 hatch로 시각화. 라벨은 영역 중앙에 흰 박스 위 italic으로 항상 노출.

기본 영역
13% 채움 + sun45° hatch + 라벨
사라진 영역
Dusk · 점선 외곽 · 라벨도 italic

D줌 반응 — 3단계

현재 NaverMap.jsx의 zoom 1~3 분기 그대로. 도트·라벨 노출 단계화로 가독성 유지.

ZOOM 1 — 멀리서
7px 도트 · 라벨 숨김 · 클러스터 활성
ZOOM 2 — 중간
10px 핀 + 짧은 라벨
ZOOM 3 — 가까이
12px 핀 + 풀 라벨 · 클러스터 비활성

E한 화면에서 — 시각 위계

Ember 핀(장소) > Moss 경로(흐름) > Moss 영역(시절) > 도로/지명. 시선이 자연스럽게 핀으로.
