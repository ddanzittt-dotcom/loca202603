MOTION · 4가지 원칙

01 · 반응성 (140ms)
버튼 / 칩 / 인풋 포커스. 손가락 떼는 순간 정지.
02 · 진입·퇴장 (220ms ease-out)
시트 슬라이드 / 페이지 전환 / 토스트. 끝에서 부드럽게 멈춤.
03 · 데이터 변환 (360ms)
스켈레톤 → 콘텐츠 페이드. 화면이 완성되는 모습이 보이도록.
04 · 절대 금지
화면 전체 회전·확대 / 캐릭터 바운스 / 화려한 파티클 / 무지개 그라디언트.

VOICE · 카피 톤

✗ 임시 / 가벼움 (제거)
· "준비 중이에요!", "곧 만나요 🌱"
· "지도가 발행됐어요!" (느낌표 남발)
· "데이터를 불러오는 중입니다..."
· "오류가 발생했습니다."
✓ 침착한 큐레이터 톤

· "잠시 연결이 끊겼습니다. 다시 시도하시겠어요?"
· "첫 기록을 시작하세요"
