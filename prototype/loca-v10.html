v9 → v10 — 무엇이 진짜 바뀌었는가

시각만이 아니라 정보 위계와 인터랙션 패턴까지.

A구조 변화 — v9 vs v10

화면 영역 v9 (reskin) v10 (재설계)
홈 첫 화면유저 카드 + 통계 + RESUME + MY LOGedge-to-edge dark hero + 큰 타이틀 + 큰 CTA
레벨/XP전면 노출 (게임 같음)CTA에 작은 "🔥 4일 연속"으로 흡수
최근 기록정사각 카드 그리드살짝 회전된 폴라로이드 가로 스크롤
PICKED 큐레이션4개 작은 카드 캐러셀단일 큰 매거진 카드 + 사진 위 카피
프로필아바타 + 통계 그리드 + 2버튼 + 1지도잡지 표지 + 떠 있는 아바타 + 탭 + 매거진 카드 스택
지도 에디터헤더 카드 + 필터칩 + 큰 sheet 항상95% 지도 + 작은 pill 헤더 + swipe-up sheet (peek)
선택된 핀시트 안 한 줄지도 위 떠 있는 큰 콜아웃 카드 (사진 + 메모)
핀 상세중간 떠 있는 카드풀 시트 · 위쪽 54% 사진 hero · 아래 잡지 페이지
탐색위에서 카테고리 + 행사 카드 리스트잡지 표지 hero + featured by 큐레이터 + 그리드
BottomNav화면 끝까지 닿는 바떠 있는 캡슐 (양옆 14px) + 가운데 + FAB

B5가지 핵심 인터랙션 변화

Hero가 hero가 됨 · 화면 진입 시 첫인상은 사진 + 큰 카피. 기능 리스트가 아님
Chrome 50% 감소 · 고정 헤더·검색바·필터바 → hero 위 작은 칩 또는 swipe up
CTA가 단 하나 · 화면당 가장 중요한 행동 1개에 집중. "+기록하기" 풀 너비 ember pill
잡지·다이어리 패턴 · 폴라로이드 회전 · num font 캡션 · 큰 인용 + 작가 노트 · serif italic 메모
지도가 진짜 주인공 · 95% 지도, 콜아웃은 사진+메모 떠있는 카드, 시트는 swipe up
